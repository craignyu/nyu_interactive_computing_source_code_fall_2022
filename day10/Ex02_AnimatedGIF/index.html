Animated GIFs

Press 's' to stop, 'p' to play and 'r' to reset

Available methods on a 'GIF' object:

  • img.pause(); - pauses the animation for all instances of the GIF
  • img.play(); - plays (re-starts if stopped) the animation for all instances of the GIF
  • img.setFrame(n); - jumps to the desired animation frame for all instances of the GIF
  • img.getCurrentFrame(); - returns the current frame being played (integer)
  • img.numFrames(); - returns the total number of frames contained within the GIF (integer)
